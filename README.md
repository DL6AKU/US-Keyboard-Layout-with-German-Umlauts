US-Keyboard-Layout-with-German-Umlauts
======================================
This project provides keyboardlayouts and instructions to achieve:

 - Deactivate Caps Lock
 - CapsLock + a = ä
 - CapsLock + s = ß
 - CapsLock + e = €
 - CapsLock + u = ü
 - CapsLock + o = ö
 - CapsLock + Shift + a = Ä
 - CapsLock + Shift + u = Ü
 - CapsLock + Shift + o = Ö
